# Gemini Streamlit App

This is a simple Streamlit web app that uses Google Gemini LLM to generate content from user prompts.

## Features
- Input any text prompt
- Get AI-generated responses instantly
- Clean and interactive UI with Streamlit

## How to Run
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`

## Files
- `main.py` → Gemini client setup and helper function
- `app.py` → Streamlit interface
